# Piedra, Papel o Tijera ✊✋✌️

Proyecto de práctica realizado para la asignatura de **Desarrollo Web en Entorno Cliente**.

## 🕹️ ¿Cómo funciona?

El usuario selecciona una opción (piedra, papel o tijera) y juega contra la máquina, que elige aleatoriamente. Se muestra el resultado y se actualiza el marcador.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (manipulación del DOM y eventos)

## 🎯 Objetivo del proyecto

Practicar estructuras condicionales, eventos, manipulación del DOM y lógica de programación básica en JavaScript.
